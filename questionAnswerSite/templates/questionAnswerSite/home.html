Welcome to the Question and Answer final project page.

Some features I provide.

Feature 1


1. Allow multiple users to login and view questions and modify questions.
2. Non login user can view all questions without problem. ->Look Around->All questions.
3. Only show 10 questions on a page, this is done through query. ->Look Around->All questions.
4. Answers are sorted by largest difference between up and down.->Look Around ->All questions->click one of the questions, you can see all anwsers for that question.
5. you can tag for a question when creating a question. ->Look Around ->All questions->Create link.
6. http site is showed in a link format while jpg is showed as a inline picture.
7. you can upload picture with your id. ->Look Around ->My picutres, This is done through blobstore in epp engine.

Feature 2


8. Each question and answer has a permalink, you can find it through ->Look Around ->All questions->click one of the questions->go to link. And limited in 500 Char on the home page.
9. Each question and answer timestamps are saved in its permalink, you can find it through ->Look Around ->All questions->click one of the questions->go to link.
10.RSS link is created using Django library. ->Look Around -> RSS Link.

Additional Feature


Bootstrap User interface.
